Xenial Build Containers for Enterprise (beta)

  • 使用Travis CI Enterprise 2.2.6及更高版本进行安装
  • 重新启动travis-worker

##

平台要求：要使用Xenial构建容器，Travis CI安装必须为2.2.6或更高版本。请务必升级

r />

我们建议使用具有8个vCPU和15 GiB内存以及至少40 GiB磁盘空间的计算机。如果您使用的AWS将是他们的c4.2xlarge实例类型。此外，您将要运行Ubuntu 16.04或更高版本。在安装和操作期间，端口22必须为SSH打开。

。要运行Trusty和Xenial构建，至少需要两个工作器实例。

使用Travis CI Enterprise 2.2.6及更高版本进行安装 #

$ curl -sSL -o /tmp/installer.sh https://raw.githubusercontent.com/travis-ci/travis-enterprise-worker-installers/master/installer.sh

$ sudo bash /tmp/installer.sh \
--travis_enterprise_host="[travis.yourhost.com]" \
--travis_enterprise_security_token="[RabbitMQ Password/Enterprise Security Token]" \
--travis_beta_build_images=true

重新启动travis-worker #

$ sudo systemctl restart travis-worker

在启动时应用工作程序配置更改。

##

要在新的Xenial构建环境中运行项目的构建，请添加一个dist: xenial.travis.yml
