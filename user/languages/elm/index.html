选择Elm版本进行测试 #

您可以指定Elm语言版本来测试您的项目

elm键。

这可以是单个字符串值（例如，0.19.0），或者是一个字符串数组

在哪种情况下，构建矩阵由运行每个版本的作业组成

在数组中指定。

language: elm
elm:
  - elm0.19.0 # default, or equivalently, '0.19.0'
  - elm0.18.0

支持工具elm-test### 和elm-format #

榆树工作也将安装elm-test和elm-format。

默认情况下，这些工具对应于elm版本指定，用

字首elm。

例如，用elm: 0.19.0，版本elm-test和elm-format

与标签elm0.19.0将会被安装。

elm-test和elm-format密钥独立：

language: elm
elm-test: 0.19.0-rev3
elm-format: 0.8.0

{：data-file =“。travis.yml”}

请注意，这些值将传递给npm。

这意味着可以将它们指定为版本字符串（例如，0.19.0）

或标签（例如，elm0.19.0）。

elm-test和elm-format默认为标签elm前缀，源自

elm值。

###

Elm项目也可以指定Node.js版本

Node.js的

language: elm
node_js: '10' # latest 10.x

默认的Node.js版本是10.13.0。
