本指南涵盖的内容 #

本指南介绍了特定于Nix项目的构建环境和配置主题。请务必阅读我们的教程和一般构建配置首先指导。

社区支持的警告 #

或随时改变。如果您遇到任何问题，请在报告中报告

概观 #

要安装Nix存储并设置基本单用户配置文件，请设置language键入.travis.yml至nix。

language: nix

{：data-file =“。travis.yml”}

默认通道nixpkgs将会nixpkgs-unstable。

提供工具 #

Nix环境中提供了以下命令行工具：

  • 尼克斯
  • 尼克斯建造
  • 尼克斯沟道
  • 尼克斯 - 收集 - 垃圾
  • 尼克斯拷贝闭合
  • 尼克斯守护
  • 尼克斯-ENV
  • 尼克斯实例化
  • 尼克斯预取的URL
  • 尼克斯壳
  • 尼克斯店

默认Nix版本 #

这将安装Nix 2.0.4使用https://nixos.org/releases/nix/nix-2.0.4/install。您可以使用指定不同版本的Nix安装程序nix:关键在你的.travis.yml：

language: nix
nix: 2.0.4

{：data-file =“。travis.yml”}

注意：此选项支持所有Nix版本，从版本1.11.16开始。

默认目标 #

默认的构建脚本是nix-build它构建了一切default.nix存储库根目录的文件。这可以通过设置来覆盖script关键在于.travis.yml文件。例如，

language: nix
script: nix-build -A tarball release.nix

{：data-file =“。travis.yml”}

上面的配置将尝试从release.nix中的Nix表达式构建属性“tarball”。
