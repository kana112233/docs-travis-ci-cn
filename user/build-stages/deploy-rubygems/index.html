Build Stages: Deploying to Rubygems

这个例子有2个构建阶段：

  • 两个分别针对Ruby 2.2和2.3运行测试的作业
  • 将gem发布（发布）到rubygems.org的一项工作

这是什么的.travis.yml配置可能看起来像：

rvm:
  - 2.2
  - 2.3

script: echo "Running tests against $(ruby -v) ..."

jobs:
  include:
    - stage: gem release
      rvm: 2.2
      script: echo "Deploying to rubygems.org ..."
      deploy:
        provider: rubygems
        gem: travis-build-stages-demo
        api_key: $RUBYGEMS_API_KEY
        on: deploy-gem-release

{：data-file =“。travis.yml”}
