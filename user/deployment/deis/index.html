Deis Deployment

Travis CI支持上传到DEIS。

最小配置是：

deploy:
  provider: deis
  controller: deis.deisapps.com
  username: "Deis User Name"
  password: "Deis Password"
  app: App_name
  cli_version: vX.Y.Z  # e.g. v2.7.0 being the latest at this time

{：data-file =“。travis.yml”}

建议您加密密码。

假设您安装了Travis CI命令行客户端，您可以这样做：

$ travis encrypt "YOUR DEIS PASSWORD" --add deploy.password

系统将提示您在命令行中输入api密钥。

你也可以travis工具为您设置一切：

$ travis setup deis

请记住，上面的命令必须在项目目录中运行，因此它可以修改.travis.yml为了你。

在部署之前和之后运行命令 #

有时您希望在触发部署之前或之后运行命令。你可以使用before_deploy和after_deploy这个阶段。只有在特拉维斯CI实际推动发布时才会触发这些。

    before_deploy: "echo 'ready?'"
    deploy:
      ..
    after_deploy:
      - ./after_deploy_1.sh
      - ./after_deploy_2.sh

{：data-file =“。travis.yml”}
