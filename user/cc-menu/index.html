Using CCMenu with Travis CI

CCMenu是macOS状态栏的一个小工具，可以从Mac的便利性中跟踪存储库的最新构建状态。

是Windows环境的等效工具，BuildNotify对于Linux系统。一般说明适用于所有这些说明。

它们最初是为与CruiseControl一起使用而构建的，但它们与Travis CI一样好用，您可以使用它们来轮询您的Travis CI存储库并获得其状态

显示在菜单栏或托盘中。

将CC提要与存储库一起使用 #

开源存储库使用URL方案https://api.travis-ci.org/repos/<owner>/<repository>/cc.xml访问CruiseControl提要。它们直接来自我们的API。

要添加存储库源，我们提供生成的URL（适用于公共和私有存储库），可以通过访问存储库页面并单击Build Status Image来访问它，如下所示：

打开模态后，选择CCTray从第二次下拉，这

然后显示要复制的URL。如果存储库是私有的，它也会

包括必需的令牌参数：

必须具有以下形式：

  • 对于开源项目使用https://api.travis-ci.org/repos/<owner>/<repository>/cc.xml?branch=<branch>
  • 对于闭源项目使用https://api.travis-ci.com/repos/<owner>/<repository>/cc.xml?token=<token>&branch=<branch>

###

上述技术只允许您一次添加一个存储库，对于拥有多个存储库的组织的团队成员来说，这可能很难处理。您可以只指定所有者并选择项目的子集，而不是指定所有者和存储库。

  • https://api.travis-ci.org/repos/<owner>.xml
  • 对于闭源项目使用https://api.travis-ci.com/repos/<owner>.xml?token=<token>。

CCMenu将向您显示所有可用存储库的列表，然后您可以在单个快速操作中添加这些存储库。
