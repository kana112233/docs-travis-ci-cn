本指南涵盖的内容 #

本指南介绍了Travis Windows CI环境中可用的软件包，工具和设置（通常称为“CI环境”）。

请注意，我们的Windows环境处于早期阶段，目前支持Linux或macOS上可用的最小部分。

支持 #

我们的Windows环境的早期采用者可以在其中询问他们的问题/报告问题Windows类别我们的社区论坛

使用Windows #

要使用我们的Windows构建基础结构，请将以下内容添加到您的.travis.yml：

os: windows

{：data-file =“。travis.yml”}

Windows版本 #

只要Windows Server，版本1803目前支持。

巧克力味 #

巧克力味，Windows的程序包管理器已安装，可用于安装Windows程序包。

Git BASH #

Git BASH是用于运行构建的shell。看到Git for Windows更多细节。

电源外壳 #

可以通过调用来使用Powershellpowershell在你的.travis.yml文件中。我们正在考虑很快添加一流的Powershell支持。

文件系统 #

运行Windows的VM使用默认文件系统NTFS。

支持的语言 #

  • C用language: c
  • C ++与language: cpp
  • Node.js用language: node_js
  • 生锈language: rust
  • 跟着去language: go
  • 巴什language: bash要么language: shell
